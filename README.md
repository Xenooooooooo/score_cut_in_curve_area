# 圈定区域裁剪工具

这是一个 Python 桌面小工具：运行后选择图片文件夹，按文件名顺序逐张读取图片，在画布上圈定区域，并将圈定区域内非白色像素的边界裁剪输出为 JPG。

## 安装

```powershell
python -m pip install -r requirements.txt
```

## 运行

```powershell
python main.py
```

启动后会弹出文件夹选择窗口。选择完成后，程序读取该文件夹内的常见图片格式：`jpg`、`jpeg`、`png`、`bmp`、`gif`、`tif`、`tiff`、`webp`。

## 操作

- 鼠标滚轮：缩放图片
- 普通模式下拖动画布：移动图片
- `圈定区域` 或 `Q`：进入圈定模式。第一次单击图片开始圈定，移动鼠标绘制曲线，第二次单击完成闭合
- `撤销` 或 `Ctrl+Z`：清除当前圈定区域
- `带边界区域裁剪` 或 `E`：按圈定区域内的非白色像素边界裁剪，并补白边输出
- `下一张` 或 `右箭头`：切换到下一张图片

输出文件会保存在所选图片文件夹下的 `out` 文件夹中。比如 `atime.jpg` 在同一轮操作中裁剪三次，会依次输出为 `atime_1.jpg`、`atime_2.jpg`、`atime_3.jpg`。

## 裁剪规则

程序以原图左上角为 `(0, 0)`，向右为 X 正方向，向下为 Y 正方向。点击裁剪时，会在圈定区域内寻找所有非白色像素，并得到它们最上、最下、最左、最右的边界。

随后计算：

- 最左非白色点到图片左边界的距离
- 最右非白色点到图片右边界的距离

取两者较小值作为白边宽度，在裁剪出的矩形区域上下左右各添加同等宽度白边后，以 JPG 格式输出。
